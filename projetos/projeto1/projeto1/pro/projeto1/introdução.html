Sobre o Projeto

O Projeto Gc Tech foi desenvolvido para atender às necessidades da empresa Gc Tech Consultoria, especializada em DevOps, Cloud, APM, Desenvolvimento de Software e AWS.
O site foi totalmente personalizado, seguindo as especificações do proprietário da empresa. Trabalhamos em conjunto, analisando três modelos de concorrentes, para criar um esboço no Figma. Em seguida, desenvolvi o site usando HTML e CSS, garantindo que fosse responsivo e atendesse às expectativas do cliente.
Este projeto foi fundamental para o meu crescimento profissional.
